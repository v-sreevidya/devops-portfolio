## Linux Tools
## tcpdump
- tcpdump -> used to analyse network traffic
- tcpdump -c 100 -> this only shows the traffic and packet analysis of 100 packets.

## htop
- It is used for monitoring and analysing the memory, CPU, load average, uptime etc.
- It is very useful for debugging as it gives the overall statistics of the VM

## strace
- It is used to trace the process that is happening in the backend when a command is run

## netstat
- It basically tells the connections and their routing along with the IPs that are there from this VM (network connections)
- Live data available

## dmesg -T
- This shows kernel logs (important when OOM hits)
